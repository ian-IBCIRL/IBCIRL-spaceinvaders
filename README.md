# 🚀 Space Invaders - Strategic Mobile Game

A classic Space Invaders game with a strategic twist, built with vanilla JavaScript and optimized for all devices - mobile, tablet, and desktop.

## 🎮 Play Online

[Play the game here](https://ian-IBCIRL.github.io/IBCIRL-spaceinvaders) 

## 🎯 Strategic Gameplay

Unlike traditional Space Invaders, this version features **strategic scoring**:
- **Each shot costs 5 points** - no more spray and pray!
- **Each hit rewards 10 points** - precision is rewarded
- **Live accuracy tracking** - see your skill improve over time
- **Performance stats** - compare your accuracy with other players

## 🏆 Scoring System

### Strategic Scoring
- **Hit an invader**: +10 points
- **Fire a shot**: -5 points
- **Net gain per hit**: +5 points
- **Missed shots**: -5 points (pure loss)

### Performance Metrics
- **Accuracy**: Percentage of shots that hit targets
- **Efficiency**: Higher accuracy = higher scores
- **Skill tracking**: See improvement over multiple games

### Example Scores
- **90% accuracy** (36/40 hits): 360 - 200 = **160 points**
- **50% accuracy** (20/40 hits): 200 - 200 = **0 points**
- **Perfect game** (40/40 hits): 400 - 200 = **200 points**

## 🚀 Quick Start

### Play Instantly
Just visit the [live demo](https://ian-IBCIRL.github.io/IBCIRL-spaceinvaders) and start playing!

### Run Locally
1. **Clone the repository**
   ```bash
   git clone https://github.com/ian-IBCIRL/IBCIRL-spaceinvaders.git
   cd IBCIRL-spaceinvaders
   ```

2. **Open in browser**
   - Open `index.html` in any modern web browser
   - No build process required!

## 📱 Install as Mobile App

On mobile devices, you can install this as a Progressive Web App:

1. Open the game in your mobile browser
2. Look for "Add to Home Screen" option
3. Install and play like a native app!
